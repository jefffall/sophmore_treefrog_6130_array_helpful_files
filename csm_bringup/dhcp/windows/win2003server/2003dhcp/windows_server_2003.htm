Windows Server 2003

Setting up a DHCP server:

 

  • From Manage Your Server, click Add or remove a role. By default, Manage Your Server starts automatically when you log on. To open Manage Your Server, click Start, click Control Panel, double-click Administrative Tools, and then double-click Manage Your Server.
  • To open the Configure Your Server Wizard, click Start, click Control Panel, double-click Administrative Tools, and then double-click Configure Your Server Wizard.

 

 

Select an unused range of addresses from your network

No exclusions needed. Leave blank.

Add a working gateway on your subnet. By convention, gateways sometimes end in a 1.

 

Don't need or care about DNS for the array. Leave blank.

 

Activate the scope later as BOOTP information must be added.

Reopen the DHCP role

Expand it.

Add reservations to mate the MAC address of the A and B array controllers to a BOOTP IP address.

Use ADD again to add this next reservation. That means right clicking on reservations again.

 

Verify the reservations are there, the MAC addresses are correct for controllers A and B respectively.

 

Bring up the scope. Right click on it and start it.

The scope will go from red to blue.

Look at the address leases. If the array performs BOOTP on  given IP addresses, the addresses under reservations will state transition from inactive to active.

Close the Manager Your Server window.
