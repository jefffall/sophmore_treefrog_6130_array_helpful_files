6130 array default assigned IP - using from Windows 2000 Advanced Server

 

Open up control panel

Select Network and Dialup connections:

Select Local Area connection:

Select Properties

Select TCP/IP Properties

Assure a static IP address is configured. A dynamic address will not work:

Click on Advanced:

Highlight the IP address and click on the TOP Add

 

Type in the IP address and mask as shown:

Click add and note the new IP address added to the configuration:

Bring up a command shell and attempt to ping the array:

If the array pings back, then bring up Element Manager and add the IP address of the array

Choose the 6130 Configuration Service

Register the array:

Choose Register the array

Make sure the password is right...

This is as far as we can go with Element Manager Build 13 July 29th 2004
